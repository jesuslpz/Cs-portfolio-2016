Hardware

Motherboard

The motherboard is often thought to be one of the most important parts of the computer. A motherboard is a piece of hardware in the computer that connects all other pieces of hardware in that computer together. Before there were motherboards, computers used gears connected by a backplane that had places to put in other wires. The main job of a mother board is to connect all other pieces of hardware to the microprocessor chip that it holds.


CPU

Federico Faggin was hired at Intel in the April of 1970 which later resulted in the invention that is the CPU. CPU stands for the Central Processing Unit. CPU is where the computer does its primary work. Every single computer must have some sort of CPU to function. The CPU is often referred to as the brain of the computer. The CPU handles instructions given to it by the user. The CPU uses two types of storage: primary and secondary. Also, the CPU relies on the primary storage for instructions and data.


RAM

RAM was invented by Robert Dennard. RAM stands for Random Access Memory. Computer and human memory have different purposes and operate in different ways. Computer memory or RAM stores input data, programs, and other information temporarily. It can be read or written. More RAM means more multitasking for the computer. RAM is measured in megabytes or gigabytes. I have hope that it will be measured in terabytes in the future. Like Human memory RAM is “forgotten” when the computer turns off.


Hard Drive

The Hard Drive is the memory of the computer that was invented over 50 years ago. Inside the hard drive there is a large magnetic material called a platter that is divided into a billion small areas. This is where the magnetism comes into the picture. Each small area is either magnetised to be 1 or demagnetized to be 0. That combination of zeros and ones make the memory.


BUS

BUS is a communication system that transfers data from computer to computer or even within the computer itself. The bus width is also very important. The wider the bus then the more the data can be sent at a time.


Keyboard, Mouse, and Monitor

A keyboard is a device that allows people to type letters, number, and symbols in the computer. Qwerty is the most popular type of keyboard, since it was especially designed to make typing easier. Each key has a little electrical switch that sends information to the computer. The computer can then tell which keys were pressed and in what order. A mouse uses a light and a light detector to track the movement of the mouse and apply it to the cursor on the monitor. Steve Kirsch of MIT invented a type of mouse that is commonly used today. A monitor is pretty much a screen like the screen on a laptop or phone. The outside of the screen is usually protected by fiber optic glass. All of these work together to make the experiences of computer more simple and enjoyable for the user.
