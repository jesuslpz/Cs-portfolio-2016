First, I wanted to make the submit button work. So I used the DOM addEventListener which "listens" for the click. When the button is clicked it performs the function. Next, I added "<br>" to the function which breaks the text and makes more space. I chose to add this, because I did not want the tasks too close to eachother and hard to read. After that I added the "<div style='color:" to be the ColorInmput.value. I did that so the user could actually change the color of the tasks. Clearing Tasks 


reset.addEventListener("click" , function(evt){
    tasks.innerHTML = ""
}
tasks.addEventListener("click", function(evt) {

     var targetTasks = evt.target;
     targetTasks.parentNode.removeChild(targetTasks);

 });

First, I used addEventListener to "listen" for a click in the variable reset. When the click is heard, it runs a specific function. This function first uses tasks.innerHTML to manipulate the display of the website. Then it sets it to nothing. There is nothing in the quotes, because I wanted it to clear everything. So it is not really clearing everything as much as it is replacing it with nothing. Next, I used another addEventListener, but this time I made it so it will recognize when each individual tasks is clicked by using tasks instead of reset. When a task is clicked, the variable targetTasks targets an idividual tasks and not all of them. Then, the targetTasks.parentNode.removeChild(targetTasks); is what removes the individual task. I added this into my todo list so that if I make a mistake I don't have to clear all of my tasks.
