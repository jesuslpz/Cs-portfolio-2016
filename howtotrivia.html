How to: Trivia

Overview of Quiz

As obvious as it may be, we decided to create a trivia quiz. After thinking over what it could be about, we decided that a really random quiz would be fun and easy. Next, came the hard part actually making it. Creating the questions was not too difficult. All we did was to use a question code format that Aakaash found and create one for each question. The format we decided used radio style buttons which was really perfect for what we were trying to do. Finally, at the end of the html page we had a Submit answer button. This button would not be very useful had it not been for the code right under it. Then right under is the html piece we used to achieve the answer score at the bottom. We inclosed information on what we wanted it to do: score. I lied there is one more part under that. The last piece to our HTML page was code that linked our HTML page to our Javascript page.

How to Determine Correct Answer

Now here comes the interesting part. After the format, it time to put all that into effect with javascript. We want the computer to know which answer is correct. We gave the wrong answers a value of zero. That was the first step to determining the right answer. Then, we not only gave the correct answers values of one, but we also put them all in one class, the correct class.
