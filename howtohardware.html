How To

Links

In the picture on the right you can see the code that I used to style my links. Also, you can see the end result on left in a red circle. In the code, the a represents the links. I changed the color of the links to pink with the css key “color” and the value “pink”. To change it any other color all I would have to do is keep the key of color the same and change the value to something other than pink. Then, I used the “text-decoration” key to “decorate the text.” I set the value to “underline” to underline all of the links.

Background

This coding was one of my favorites, because it worked great and was easy to use. I used the body tag that defines an element's body. Then I used background-image property to change the background image. I also used url to let the system know I was about to write the image address. Finally, I ended it with a bracket to let the system know that was it.

Heading

This code was one that I used to put in the title of the Hardware page. I started with h1, since this was the first heading. Then I used an opening font tag to manipulate the way the heading will look. I changed the size to 30, because it was the title of the page. I also then changed the color to hotpink. Then I wrote the text that was going to show up on the page. Then I typed the closing font tag and the closing h1 tag.

Navbar

To start coding the navbar I used an opening unordered list tag. Also, I made the id equal to navbar which makes it so I can edit the navbar appearance and such on a navbar different page. Then, I used an opening list item tag to open each of my list items. After that, I used an a tag to introduce the links. The links are what make it possible for me to switch between the pages I made. I then closed with a closing link and closing list item tag. Also, I have placeholder “Hashtags” which are there to hold the place for the links I will add later. Finally, I end it all with a closing unordered list tag.
